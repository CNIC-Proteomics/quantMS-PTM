Start the nextflow container
```

```



Remove a container
```
docker rm nextflow
```

Remove an image
```
docker rmi nextflow
```


# Compose PTM-compass project
-------------------------

From your project directory, start up your application by running
```
docker-compose up
```

Run the ptm-compass service
```
docker compose run ptm-compass
```
